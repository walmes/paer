Download e Instalação do R e Editores

1 Download e Instalação do R

O software de computação estatística R ésta disponível para download no repositório oficial do projeto R: https://www.r-project.org/.

A instalação do R para usários dos sistemas operacionais Linux, Mac OS X e Windows, está descrita em: http://cran-r.c3sl.ufpr.br/.

Usários do SO Windows podem instalar o R a partir do executável *.exe disponibilizado em: http://cran-r.c3sl.ufpr.br/bin/windows/. Para uma instalação padrão, basta clicar em avançar durante as etapas até chegar em concluir. Usuários de Mac OS X devem seguir as instruções diponíveis no site.

Para os usuários de Linux, existem tutoriais para instalação em 4 distribuições: Debian, Redhat, Suse e Ubuntu. Os passos para instalação do R no Ubuntu estão descritos abaixo.

Abra um terminal no seu Ubunto (atalho Ctrl + Alt + t). Em seguida verique a versão do Ubuntu que está usando.

# Mostra a versão do Ubuntu.
lsb_release -a
No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 16.04.1 LTS
Release:    16.04
Codename:   xenial

Em seguida, abra o arquivo /etc/apt/sources.list como super usuário.

# Abre o arquivo como super usuário.
sudo gedit /etc/apt/sources.list

Adicione a linha abaixo no final do arquivo. Salve, feche e volte ao terminal.

# Modelo de repositório e distribuição Ubuntu.
deb https://<my.favorite.cran.mirror>/bin/linux/ubuntu <version>/

# Usando o repositório do C3SL para o Ubuntu Xenial Xerus (16.04)
deb http://cran-r.c3sl.ufpr.br/bin/linux/ubuntu xenial/

No termianal atualize a lista de repositórios (fetch) e instale o r-babe.

sudo apt-get update
sudo apt-get install r-base

Depois de instalar o r-base pelo terminal, você já terá o R na versão mais enxuta, ou seja, aquela com os pacotes recomentados apenas. Os demais pacotes podem ser instalados de dentro do R, com install.packages(), como será mostrado a frente.

2 Instalação do RStudio

O RStudio talvez seja o único editor específico multiplataforma para o R. O RStudio é bastante fácil de usar e por isso é recomendado para aqueles que estão começão a trabalhar com R.

Na área de downloads do site do RStudio, pode-se baixar os instaladores para Ubuntu (.deb), Windows (.exe) e MAC OS X (.dmg) além de Fedora, Redhat e Suse (.rpm).

O executável .exe do Windows é de simples instalação. Da mesma forma, com o arquivo .deb para Ubuntu, o RStudio é instalado por meio de interface gráfica.

Pode-se trabalhar no R com outros editores. A lista abaixo

  1. Emacs com o pacote ESS (Emacs Speaks Statistis).
  2. Vim com o plugin Nvim-R.
  3. Tinn-R.
  4. Eclipse com StatET.
  5. Sublime Text.
  6. Gedit com Rgedit plugin.

Creio que os mais utilizados sejam o RStudio com os três primeiros da lista, não necessariamente nesta ordem. Viste as homepages para saber como usar estes editores para trabalhar com o R.

O Emacs1, junto com o Vi, são os editores mais antigos dessa lista conforme o texteditors.org. Como não são editores de R, pois surgiram 17 anos do R, e por serem altamente customizáveis, a maioria dos desenvolvedores no começo da história do R, e muitos até hoje, fazem uso destes editores.

3 Instalação do Emacs

Para instalar o Emacs no Ubuntu, execute cada linha do código abaixo em um terminal. O Emacs atualmente está na versão 24.5.1.

sudo apt-get install build-essential
sudo apt-get install emacs emacs-goodies-el ess
emacs --version
